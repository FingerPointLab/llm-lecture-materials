### API Key
API 키는 인증 및 추적을 위해 사용됩니다. 요청한 사용자를 구분하고, 권한을 검증하며, 로그를 기록할 수 있습니다.

### Webhook
Webhook은 서버 간 실시간 통신을 위해 사용하는 방식으로, 특정 이벤트 발생 시 자동으로 데이터를 전송합니다.

### OAuth
OAuth는 제3자 애플리케이션이 사용자 인증을 위임받아 안전하게 API에 접근할 수 있도록 하는 프로토콜입니다.

### REST API
REST API는 HTTP 프로토콜을 기반으로 한 설계 방식으로, 자원의 상태를 전송하고 조작할 수 있도록 합니다.

### JWT
JWT(Json Web Token)는 인증 정보를 JSON 형태로 저장하고 서명하여 전송하는 방식으로, Stateless 인증에 주로 사용됩니다.

### CDN
CDN(Content Delivery Network)은 전 세계에 분산된 서버를 통해 콘텐츠를 빠르게 제공하는 기술입니다.

---

### SAP 생산오더
SAP에서 생산오더를 생성하려면 CO01 트랜잭션 코드를 사용합니다. 생산오더 생성 시 자재번호, 수량, 공장정보를 입력합니다.

### SAP 자재이동
SAP에서 자재이동은 MIGO 트랜잭션을 통해 수행되며, 입고, 출고, 이동을 관리합니다.

### SAP 자재마스터
SAP 자재마스터(Material Master)는 자재의 기술정보, 재고정보, 구매정보 등을 포함합니다. MM01 트랜잭션으로 생성합니다.

### SAP 회계전표
SAP에서 회계전표는 BKPF 테이블에 저장되며, 회계 이벤트를 문서화합니다. FB01로 신규 전표 생성이 가능합니다.

### SAP BOM
SAP BOM(Bill of Materials)은 제품을 구성하는 자재 목록으로, CS01 트랜잭션으로 생성합니다.

---

### 보험 계약해지
보험 계약을 해지하려면 해지 사유, 해지 신청서, 신분증 사본이 필요합니다. 해지 사유는 명확해야 하며 예외 사항은 별도 심사됩니다.

### 보험 청약철회
청약철회는 보험가입 후 일정 기간 내 계약을 무효로 만드는 제도로, 계약일로부터 15일 이내에 요청해야 합니다.

### 보험금 청구
보험금 청구는 사고 발생 시 보험사에 보상금 지급을 요청하는 절차로, 사고 증빙자료와 함께 제출해야 합니다.

### 보험 계약대출
계약대출은 보험계약의 해지환급금을 담보로 일정 금액을 대출받는 제도입니다.

### 보험 실효
보험료를 납부하지 않으면 보험계약은 실효되어 보장 효력이 상실됩니다. 일정 기간 내 연장납입으로 복원 가능합니다.
